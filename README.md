# Komiwojazer-c++
Program tworzy automatycznie graf pełny dla podanej liczby wierzchołków (domyślnie 15) i przy użyciu algorytmu genetycznego szuka rozwiązania dla problemu komiwojażera o jak najbardziej optymalnej (najmniejszej) wartości funkcji celu.
Algorytm został stworzony żeby działać na dużych instancjach problemu, najlepiej więcej niż 10 wierzchołków.
